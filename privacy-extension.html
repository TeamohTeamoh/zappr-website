Privacy Policy

Zappr QR Scanner Extension

Effective as of 2026-03-04

This extension does not collect, store, transmit, or share any personal data or browsing information.

Camera Usage

The extension only uses the device camera to scan a QR code displayed by the Zappr iOS app. The scanned QR code contains a local address which is then opened in a new browser tab to allow communication with the Zappr app on the same local network.

Data Collection

  • No images from the camera are stored or transmitted.
  • No user information, browsing data, or usage statistics are collected.

Local Processing

All processing happens locally on the user's device.
